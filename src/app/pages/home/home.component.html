Información del Sistema

  • Frontend: Angular 11
  • Backend: Laravel 8
  • Base de Datos:MySql
Angular

Se utilizó este Framework de JavaScript por su manera dar una experiencia más fluida a los usuarios, generando aplicaciones de una sóla página muy versátiles.

Laravel

Se utilizó este framework de PHP debido a su sintaxis refinada y expresiva para crear código de forma sencilla. Es una herramienta de trabajo muy robusta para desarrollar APIS REST.

Base de Datos

Se utilizó Mysql debido a su compatibilidad con el lenguaje de programación en que se desarrolló la API. Además, su sistema de gestión de bases de datos relacional es muy potente.
